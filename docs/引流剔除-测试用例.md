# 引流剔除测试用例

> **关联**：[需求文档（PRD）](./引流剔除-需求.md) · [架构设计](./引流剔除-架构设计.md)  
> **金字塔**：P0 手工冒烟 → P1 页面源码/元数据 → P2 文档与配置 → P3 回归（核心功能）

---

## 0. 测试环境

| 项 | 值 |
|----|-----|
| 项目路径 | `qiaomu-app-review-insights/` |
| 本地启动 | `npm install && cp .env.example .env.local && npm run dev` |
| 测试 URL | `http://localhost:3000` |
| 前置条件 | `.env.local` 中**不设置** `NEXT_PUBLIC_UMAMI_WEBSITE_ID`（验证默认无统计） |
| 详情页样本 | 任选一个已缓存 App，或搜索 ChatGPT 生成后访问 `/apps/cn/{appId}/...` |

**全局禁止出现的字符串**（页面 HTML / 响应 JSON，除非在测试「改造前基线」对照组）：

```
tuijian.qiaomu.ai
github.com/joeseesun
x.com/vista8
/qiaomu/reward
/qiaomu/wechat
umami.qiaomu.ai
author.url.*qiaomu.ai
```

## 1. P0 — UI 强引流（必测）

### TC-P0-01 首页页脚无外链

| 项 | 内容 |
|----|------|
| 步骤 | 1. 打开 `http://localhost:3000`<br>2. 滚动至页脚 |

| 优先级 | P0 |

### TC-P0-02 首页头部无打赏/公众号入口

| 项 | 内容 |
|----|------|
| 步骤 | 1. 打开首页<br>2. 检查 header 右侧 |
| 期望 | 无礼物图标（打赏）、无二维码图标（关注公众号） |
| 优先级 | P0 |

### TC-P0-03 App 详情页页脚

| 项 | 内容 |
|----|------|
| 步骤 | 1. 打开任意 `/apps/{country}/{appId}/...` 详情页<br>2. 检查页脚 |
| 期望 | 同 TC-P0-01 |
| 优先级 | P0 |

### TC-P0-04 打赏/微信静态资源不可访问

| 项 | 内容 |
|----|------|
| 步骤 | 1. `curl -I http://localhost:3000/qiaomu/reward.png`<br>2. `curl -I http://localhost:3000/qiaomu/wechat.jpg` |
| 期望 | HTTP 404（或资源已删除） |
| 优先级 | P0 |

### TC-P0-05 限流不弹打赏（首页搜索）

| 项 | 内容 |
|----|------|
| 前置 | 将 `APP_REVIEW_PUBLIC_DAILY_NEW_APP_LIMIT=0` 写入 `.env.local` 并重启；或使用新 IP/清 `.generation-guard` 数据后快速耗尽配额 |

| 期望 | 出现错误/提示条（如「今天的新 App 生成次数已用完…」）；**不出现**打赏 Dialog / 二维码 |

### TC-P0-06 限流不弹打赏（榜单生成）

|----|------|

| 步骤 | 1. 首页榜单区点击某 App「生成洞察」 |
| 期望 | 同 TC-P0-05 |
| 优先级 | P0 |

### TC-P0-07 页脚无个人宣传文案

| 项 | 内容 |
|----|------|
| 步骤 | 查看首页/详情页页脚文本 |

## 2. P1 — SEO / 元数据 / API

| 优先级 | P1 |

## 7. 自动化建议（后续）

可将 TC-P1-08 加入 CI：

```bash
#!/bin/sh
set -e
PATTERN='tuijian\.qiaomu|github\.com/joeseesun|x\.com/vista8|/qiaomu/reward|/qiaomu/wechat|umami\.qiaomu'
if rg -q "$PATTERN" src public 2>/dev/null; then
  echo "FAIL: traffic-driving content detected"
  rg -n "$PATTERN" src public
  exit 1
fi
echo "PASS: no traffic-driving patterns in src/public"
```

可选扩展：对 `README.md`、`package.yml` 做同样扫描。
