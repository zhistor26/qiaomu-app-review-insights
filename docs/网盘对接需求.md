## 2. 需求目标

### 2.1 必须达成（P0）

1. 用户在分析完成后，可将 **Markdown 分析报告** 保存到 **懒猫网盘**（非仅本地下载）。
2. 保存时弹出懒猫官方文件选择器（本地 / 懒猫微服 二选一，选懒猫即进网盘）。
3. 保存成功后，用户可在网盘对应目录看到 `.md` 文件，内容与 App 内分析一致。
4. `package.yml` / 商店描述中写明「支持导出报告到懒猫网盘」。

### 2.2 明确不做（Out of Scope）

- ❌ 网盘批量导入 App 列表
- ❌ 网盘右键打开文件（`file_handler`）
- ❌ 定时任务自动写网盘
- ❌ 评论 CSV 导出到网盘（第二期再说）
- ❌ 重写 storage / 数据迁网盘
- ❌ PDF 导出、分享链路、客户端 SDK

## 3. 用户故事

**作为** 微服用户  
**我希望** 分析完 App Store 评论后，一键把报告存进家庭网盘  
**以便** 备份、分享、在其他设备查看，且应用符合懒猫商店上架要求  

**验收**：从点击到网盘出现文件，全程 ≤ 2 分钟，无需命令行。

## 4. 功能说明

### 4.1 入口

在已有「导出 / 下载报告」位置增加或替换为：

| 位置 | 文件 | 动作 |
|------|------|------|
| 分析仪表盘 | `src/components/analysis/analysis-dashboard.tsx` | 按钮：**「保存至懒猫网盘」** |
| 报告预览（可选，与上一重复则只改一处） | `src/components/report/report-preview.tsx` | 同上 |

> **最小实现**：只改 `analysis-dashboard.tsx` 一处即可。

### 4.2 交互流程

```
用户完成分析
  → 点击「保存至懒猫网盘」
  → 前端请求 GET /api/apps/{id}/export-report?format=markdown（已有 API）
  → 拿到 Blob
  → 调用 showSaveFilePicker（被 manifest inject 拦截）
  → 用户选「懒猫微服」→ 选目录 → 确认
  → 网盘出现 {AppName}-分析报告-{日期}.md
```

### 4.3 技术方案（推荐：inject，改动最少）

**manifest 层**（`lzc-manifest.yml`）：

- 增加 `application.injects`，加载 `lzc-file-chooser-inject.js`
- 拦截 `showSaveFilePicker`

**构建层**（`lzc-build.yml`）：

- 将 inject 脚本打入 LPK `contentdir`（或 `public/lazycat-injects/`）

**前端层**：

- 导出按钮改为 `showSaveFilePicker` + 写入 Blob（**不要用** `<a download>`，inject 拦不住）

**后端**：

- **不改**。复用现有 `export-report` API。

## 5. 非功能需求

| 项 | 要求 |
|----|------|
| 兼容 | 懒猫微服 Web 客户端内浏览器 |
| 失败提示 | 用户取消选择器 → 静默或 toast「已取消」；API 失败 → 「导出失败，请先完成分析」 |
| 性能 | 报告一般 < 500KB，保存 < 5 秒 |
| 安全 | 不新增密钥；报告不含 API Key |

## 6. 验收标准（提审 / 领红包前必过）

### 6.1 功能验收

- [ ] 安装 LPK 到微服，应用可正常打开
- [ ] 添加至少 1 个 App 并完成分析
- [ ] 点击「保存至懒猫网盘」，出现「本地 / 懒猫微服」选择
- [ ] 选懒猫网盘并保存成功
- [ ] 网盘中可见 `.md` 文件
- [ ] 打开文件，含应用名、评论数、情感分析等章节（非空、非乱码）

### 6.2 上架合规

- [ ] `package.yml` 描述含网盘导出能力说明
- [ ] 已配置免密登录（inject 或 OIDC，与网盘 inject 可合并规划）
- [ ] 应用名称 / package id / icon 已改为 fork 自有品牌（非乔木原版）
- [ ] 根目录保留 MIT `LICENSE`（含原版权）

### 6.3 留证（申请激励用）

截图保留：

1. App 内「保存至懒猫网盘」按钮
2. 懒猫 / 本地选择弹窗
3. 网盘内已保存的 `.md` 文件
4. 文件内容预览
5. 开发者工具 Public Keys / 应用安装成功页（可选）

## 9. 风险与对策

| 风险 | 对策 |
|------|------|
| inject 未生效，只存本地 | 确认 inject 脚本路径、build 是否打入 LPK；按钮必须用 `showSaveFilePicker` |
| 无分析数据，导出 500 | 按钮在无分析结果时 disabled + 提示 |
| 与原版 package id 冲突 | fork 必须换 `package.yml` 里的 `package` 字段 |
| 审核认为功能重复 | 改应用名、描述、icon，并在描述中强调「网盘报告归档」 |
